# 👟 Shoes E-Commerce Website

## 📌 Overview

This project is a **Shoes E-Commerce Website** built using **EJS, CSS, and JavaScript**, focusing on dynamic rendering and user-friendly UI for an online shopping experience.

---

## 🎯 Purpose

* Practice **dynamic frontend rendering using EJS**
* Build a **real-world e-commerce UI**
* Improve **JavaScript and UI development skills**

---

## ⚙️ Tech Stack

* EJS (Embedded JavaScript Templates)
* CSS
* JavaScript

---

## ✨ Features

* 🛍️ Product listing page
* 👟 Shoes display with details
* 🛒 Add to cart functionality (if implemented)
* 🎨 Clean and responsive UI
* 🔄 Dynamic rendering using EJS
